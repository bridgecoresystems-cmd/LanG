# Сборка терминала на макетной плате (Breadboard)

**Пошаговая инструкция для прототипа на 830-точечной макетной плате**

---

## Что понадобится перед сборкой

- Макетная плата 830 точек (half+ size, минимум 30 строк)
- Провода Dupont: папа-папа, мама-папа, мама-мама
- Мультиметр (желательно, для проверки)
- Отвёртка (для регулировки контрастности LCD)

---

## Как устроена макетная плата

```
                   ЛЕВАЯ ШИНА       ПРАВЫЕ КОЛОНКИ      ПРАВАЯ ШИНА
                    +  -            f  g  h  i  j        +  -
                    |  |            |  |  |  |  |        |  |
Строка 1 ─────── [ +][-]  [a][b][c][d][e]  [f][g][h][i][j]  [+][-]
Строка 2 ─────── [ +][-]  [a][b][c][d][e]  [f][g][h][i][j]  [+][-]
...

ЛЕВЫЕ КОЛОНКИ: a b c d e  (одна группа — 5 лунок соединены горизонтально)
ПРАВЫЕ КОЛОНКИ: f g h i j (другая группа — 5 лунок соединены горизонтально)
ШИНЫ ПИТАНИЯ: + и - идут ВЕРТИКАЛЬНО вдоль всей платы
ЦЕНТРАЛЬНЫЙ РАЗРЫВ: a-e и f-j НЕ соединены между собой — туда вставляются микросхемы/ESP32
```

**Главное правило:** лунки в одной строке (например, a1-b1-c1-d1-e1) соединены между собой. Через центральный разрыв — нет.

---

## Порядок сборки

```
Этап 1: Шины питания
Этап 2: ESP32 на плату
Этап 3: Схема LEDs (зелёный, красный, жёлтый)
Этап 4: Схема Buzzer (транзистор + зуммер)
Этап 5: Level Converter для LCD
Этап 6: RC522 (Dupont-провода)
Этап 7: LCD 2004A (Dupont-провода)
Этап 8: Keypad 4x4 (Dupont-провода)
Этап 9: Питание от БП
```

---

## ЭТАП 1 — Шины питания

Используй перемычки Dupont (папа-папа) чтобы соединить левую и правую шины:

```
Левая шина           Правая шина
   +  ───────────────────  +     (красный провод)
   -  ───────────────────  -     (чёрный провод)
```

Пока питание НЕ подключай — только после полной сборки!

---

## ЭТАП 2 — ESP32 на макетную плату

ESP32 DevKit v1 вставляется поперёк центрального разрыва:

```
МАКЕТНАЯ ПЛАТА (вид сверху)

       a  b  c  d  e     f  g  h  i  j
  1   [3V3][ ][ ][ ][ ]  [ ][ ][ ][ ][GND]   ← ESP32 верхний ряд
  2   [EN][ ][ ][ ][ ]   [ ][ ][ ][ ][D23]   (MOSI)
  3   [VP][ ][ ][ ][ ]   [ ][ ][ ][ ][D22]   (SCL)
  4   [VN][ ][ ][ ][ ]   [ ][ ][ ][ ][TX0]
  5   [34][ ][ ][ ][ ]   [ ][ ][ ][ ][RX0]
  6   [35][ ][ ][ ][ ]   [ ][ ][ ][ ][D21]   (SDA)
  7   [32][ ][ ][ ][ ]   [ ][ ][ ][ ][GND]
  8   [33][ ][ ][ ][ ]   [ ][ ][ ][ ][D19]   (MISO)
  9   [25][ ][ ][ ][ ]   [ ][ ][ ][ ][D18]   (SCK)
  10  [26][ ][ ][ ][ ]   [ ][ ][ ][ ][ D5]   (SS/SDA RC522)
  11  [27][ ][ ][ ][ ]   [ ][ ][ ][ ][D17]   (Buzzer)
  12  [14][ ][ ][ ][ ]   [ ][ ][ ][ ][D16]   (LED желтый)
  13  [12][ ][ ][ ][ ]   [ ][ ][ ][ ][ D4]   (RC522 RST)
  14  [GN][ ][ ][ ][ ]   [ ][ ][ ][ ][ D0]
  15  [13][ ][ ][ ][ ]   [ ][ ][ ][ ][ D2]   (LED зеленый)
  16  [SD2][ ][ ][ ][ ]  [ ][ ][ ][ ][D15]   (LED красный)
  17  [SD3][ ][ ][ ][ ]  [ ][ ][ ][ ][SD1]   ⚠ Flash (не использовать)
  18  [CMD][ ][ ][ ][ ]  [ ][ ][ ][ ][SDD]   ⚠ Flash (не использовать)
  19  [ 5V][ ][ ][ ][ ]  [ ][ ][ ][ ][CLK]   ← ESP32 нижний ряд ⚠ Flash
         ^                               ^
    Левая сторона ESP32          Правая сторона ESP32
    (пины в колонку a)           (пины в колонку j)
```

**Как вставить:**
1. Найди центральный разрыв платы
2. Ориентируй ESP32 так, чтобы USB-разъём смотрел вверх (или вниз — как удобно)
3. Аккуратно надавливай равномерно на оба ряда пинов — не гни!
4. Пины должны войти в колонки **a** и **j** (или **b** и **i** — зависит от ширины модуля)

> Пины ESP32 в колонках **a** (или **b**) слева и **i** (или **j**) справа.
> Колонки **c-d-e** и **f-g-h** остаются свободными для проводов.

## ЭТАП 3 — LEDs (светодиоды)

Размести три светодиода в свободной зоне платы (например, строки 22-28):

```
       a  b  c  d  e     f  g  h  i  j
  22  [ ][ ][ ][ ][ ]   [+][R][▼][ ][ ]   ← LED зелёный   GPIO2
  23  [ ][ ][ ][ ][ ]   [ ][ ][ ][ ][ ]
  24  [ ][ ][ ][ ][ ]   [+][R][▼][ ][ ]   ← LED красный   GPIO15
  25  [ ][ ][ ][ ][ ]   [ ][ ][ ][ ][ ]
  26  [ ][ ][ ][ ][ ]   [+][R][▼][ ][ ]   ← LED желтый    GPIO16

  [+] = провод от GPIO
  [R] = резистор 220 Ом
  [▼] = LED (анод в ту же лунку что и резистор, катод в следующую лунку)
```

**Пошагово для каждого LED (одинаково для всех трёх):**

```
Строка X:
  f-лунка  → провод от GPIO (зелёный: GPIO2, красный: GPIO15, жёлтый: GPIO16)
  f-лунка  → один конец резистора 220 Ом
  g-лунка  → второй конец резистора 220 Ом
  g-лунка  → АНОД LED (+, длинная ножка)
  h-лунка  → КАТОД LED (-, короткая ножка)
  h-лунка  → провод в шину GND (-)
```

**Как отличить ножки LED:**
```
   _____
  /     \
 |       |
  \_____/
     |  |
     |  |
  длин. корот.
  анод(+) катод(-)
```

## ЭТАП 4 — Buzzer через транзистор NPN

Размести транзистор и buzzer в строках 30-34:

```
       a  b  c  d  e     f  g  h  i  j
  30  [ ][ ][ ][ ][ ]   [E][B][C][ ][ ]   ← Транзистор 2N2222 / BC547
  31  [ ][ ][ ][ ][ ]   [ ][ ][ ][ ][ ]
  32  [ ][ ][ ][ ][ ]   [ ][R][·][ ][ ]   ← Резистор 1 кОм (R → к базе B)

Buzzer подключается проводами Dupont (не на плату):
  Buzzer (+) красный → шина 5V (+)
  Buzzer (-) чёрный → лунка C (коллектор транзистора)
```

**Цоколёвка 2N2222** (плоской стороной к тебе, пины смотрят вниз):
```
  Плоская сторона ████
                  | | |
                  E B C
                  | | |
                  ↓ ↓ ↓
                GND База Колл.
```

**Цоколёвка BC547** (плоской стороной к тебе):
```
  Плоская сторона ████
                  | | |
                  C B E
                  | | |
                  ↓ ↓ ↓
                Колл. База GND
```

**Подключения:**
```
Лунка E (эмиттер) → провод в шину GND (-)
Лунка B (база)    → резистор 1 кОм → GPIO 17
Лунка C (коллектор) → провод Dupont → Buzzer (-) чёрный
                      Buzzer (+) красный → шина 5V (+)
```

## ЭТАП 5 — Level Converter для LCD

Level Converter — небольшая плата с 8 пинами. Вставь её на строки 36-39:

```
       a  b  c  d  e     f  g  h  i  j
  36  [LV][·][·][·][·]  [HV][·][·][·][·]
  37  [GN][·][·][·][·]  [GN][·][·][·][·]
  38  [L1][·][·][·][·]  [H1][·][·][·][·]   ← SDA (GPIO21 → LCD SDA)
  39  [L2][·][·][·][·]  [H2][·][·][·][·]   ← SCL (GPIO22 → LCD SCL)

LV = 3.3V (от ESP32 3.3V)
HV = 5V   (от шины питания)
GN = GND  (с обеих сторон)
L1/L2 = сторона 3.3V (от ESP32)
H1/H2 = сторона 5V (к LCD)
```

**Подключения:**

| Пин конвертера | Куда подключить |
|----------------|-----------------|
| LV | ESP32 3.3V |
| GND (LV сторона) | GND |
| LV1 | GPIO 21 (SDA) |
| LV2 | GPIO 22 (SCL) |
| HV | 5V шина питания |
| GND (HV сторона) | GND |
| HV1 | LCD SDA (мама-мама Dupont) |
| HV2 | LCD SCL (мама-мама Dupont) |

## ЭТАП 6 — RC522 (через Dupont провода)

RC522 НЕ вставляется на плату — подключается 7 проводами Dupont (мама-папа):

```
RC522 разъём:          Провод →         ESP32 GPIO
┌──────────┐
│ SDA(SS)  │ ──── синий  ──────────── GPIO 5
│ SCK      │ ──── жёлтый ──────────── GPIO 18
│ MOSI     │ ──── зелёный ─────────── GPIO 23
│ MISO     │ ──── белый  ──────────── GPIO 19
│ IRQ      │     (не подключать)
│ GND      │ ──── чёрный ──────────── GND
│ RST      │ ──── оранжевый ────────── GPIO 4
│ 3.3V     │ ──── красный ──────────── 3.3V  ← НЕ 5V!
└──────────┘
```

**Совет:** Используй разноцветные провода — потом легче разобраться.

## ЭТАП 7 — LCD 2004A (через Dupont провода)

LCD подключается 4 проводами к Level Converter (мама-мама):

```
LCD I2C разъём:        Провод →         Куда
┌──────────┐
│ GND      │ ──── чёрный  ──────────── GND шина
│ VCC      │ ──── красный ──────────── 5V шина
│ SDA      │ ──── синий   ──────────── Level Converter HV1
│ SCL      │ ──── жёлтый  ──────────── Level Converter HV2
└──────────┘
```

**Регулировка контрастности:**
На задней стороне LCD I2C модуля есть синий потенциометр.
Если экран пустой или всё белое — покрути его отвёрткой пока не появятся символы.

## ЭТАП 8 — Keypad 4×4 (через Dupont провода)

Keypad имеет 8-проводной шлейф. Подключи Dupont провода (мама-папа) в таком порядке:

```
Шлейф keypad (считай провода слева направо, когда клавиши смотрят на тебя):

Провод 1  (левый)  → GPIO 32  (Row 1)
Провод 2           → GPIO 33  (Row 2)
Провод 3           → GPIO 25  (Row 3)
Провод 4           → GPIO 26  (Row 4)
Провод 5           → GPIO 13  (Col 1)
Провод 6           → GPIO 12  (Col 2)
Провод 7           → GPIO 14  (Col 3)
Провод 8  (правый) → GPIO 27  (Col 4)
```

**Раскладка клавиш:**
```
[ 1 ][ 2 ][ 3 ][ A ]   ← Row 1
[ 4 ][ 5 ][ 6 ][ B ]   ← Row 2
[ 7 ][ 8 ][ 9 ][ C ]   ← Row 3
[ * ][ 0 ][ # ][ D ]   ← Row 4
  ^    ^    ^    ^
 Col1 Col2 Col3 Col4
```

## ЭТАП 9 — Питание от блока питания

### Подключение Barrel Jack гнезда:

```
Гнездо (вид спереди):       Провод →    Куда
┌─────────┐
│ центр + │ ──── красный ──────────── шина + (5V)
│ кольцо -│ ──── чёрный  ──────────── шина - (GND)
└─────────┘
```

### Подключение ESP32 к питанию:

```
шина + (5V)  → ESP32 VIN    (красный провод)
шина - (GND) → ESP32 GND    (чёрный провод)
```

> **Внимание:** Подключай ESP32 либо через VIN (от БП), либо через USB — но не оба одновременно!
> Для работы терминала используй только VIN от БП.

## Порядок проверки перед первым включением

### Шаг 1 — Визуальный осмотр
- [ ] Нет перемычек между соседними рядами (случайные замыкания)
- [ ] ESP32 вставлен ровно, все пины в лунках
- [ ] RC522 питание — 3.3V (не 5V!)
- [ ] LED вставлены правильно (длинная ножка к резистору)
- [ ] Транзистор ориентирован верно (E/B/C не перепутаны)

### Шаг 2 — Проверка питания мультиметром (БП подключён, USB нет)
- [ ] Шина + → 5V относительно GND
- [ ] ESP32 3.3V пин → 3.3V относительно GND
- [ ] RC522 VCC → 3.3V относительно GND

### Шаг 3 — Прошивка через USB
- [ ] Отключи БП гнездо (или отсоедини провод VIN от ESP32)
- [ ] Подключи ESP32 к компьютеру через micro-USB
- [ ] Залей скетч через Arduino IDE
- [ ] Открой Serial Monitor (115200 baud) — проверь вывод

### Шаг 4 — Запуск от БП
- [ ] Отключи USB
- [ ] Подключи VIN к шине +
- [ ] Вставь вилку БП в розетку
- [ ] Должно загореться и через 2-3 сек появиться "Tap bracelet..." на LCD

## Частые ошибки при сборке на макете

| Проблема | Причина | Решение |
|----------|---------|---------|
| ESP32 греется сразу при включении | Короткое замыкание | Отключи, проверь все соединения |
| LCD не светится | Нет питания 5V на VCC | Проверь провод к шине + |
| LCD светится, но нет символов | Неверный контраст или I2C адрес | Покрути потенциометр; запусти i2cScan |
| RC522 не видит карты в Serial | Питание 5V вместо 3.3V | СРАЗУ отключи, проверь провод VCC |
| LED не горят | Перепутан анод/катод | Разверни LED на 180° |
| Buzzer не пищит | Транзистор перепутан E/C | Проверь цоколёвку своего транзистора |
| Keypad даёт неправильные символы | Перепутаны rows/cols | Поменяй rowPins и colPins в коде |
| Нет WiFi подключения | Сеть 5 GHz | ESP32 работает только на 2.4 GHz |

## Полезные советы для макета

1. **Цветные провода** — используй систему: красный=+5V, оранжевый=3.3V, чёрный=GND. Остальные цвета для сигналов.

2. **Длина проводов** — бери короткие провода (5-10 см). Длинные путаются и дают помехи.

3. **RC522 близко к ESP32** — SPI-линии не любят длинные провода. Держи RC522 рядом.

4. **LCD подальше** — LCD большой, удобнее вынести его на отдельную плоскость рядом с макетом.

5. **Один БП или USB** — никогда не подключай БП (через VIN) и USB одновременно.

6. **Фото перед разборкой** — сфотографируй готовый макет со всех сторон, пригодится при сборке финального устройства.
