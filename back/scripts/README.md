# 🔧 Скрипты Django Backend

Эта папка содержит полезные скрипты для работы с проектом.

## Файлы:

- `fix_postgres.sh` - Скрипт для исправления настроек PostgreSQL (создание пользователя и БД)
- `setup_postgres.sh` - Скрипт для первоначальной настройки PostgreSQL
- `migrate_sqlite_to_postgres.py` - Скрипт для переноса данных из SQLite в PostgreSQL

## Использование:

```bash
# Сделать скрипты исполняемыми
chmod +x *.sh

# Запустить скрипт
./fix_postgres.sh
```
